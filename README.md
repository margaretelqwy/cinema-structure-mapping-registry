# cinema-structure-mapping-registry
本仓库用于整理影视类网站在公开访问环境中所呈现的结构映射关系，对首页栏目划分逻辑、内容层级体系、标签分布方式以及页面信息排列顺序进行系统记录。通过持续性的文字归档建立长期资料档案，用于观察不同影视平台在结构设计与信息组织方面的差异与演变趋势。所有资料均来源于公开页面，仅作为信息整理与结构研究参考使用，不包含影视资源文件，也不涉及播放或下载功能。
